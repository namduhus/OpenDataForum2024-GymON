# OpenDataForum2024
제3회 2024년 오픈데이터포럼 대회 본선 출품작-최우수상

## 출품작명: 체육ON
국민 생활체육 플랫폼
## 목적
국내 체육시설 및 체육강좌를 예약을하기 위해선 웹페이지 또는 직접방문으로 인해 불편함을 해소하고 대국민이 접근할수 있도록 앱 개발하였으며 편하게 원하는 강좌와 시설을 이용할 수 있는 예약 플랫폼 앱
